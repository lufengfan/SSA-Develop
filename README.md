# SSA-Develop

SSA-Develop包括SSA字幕文件的解析核心以及扩展的开发工具。

各个组件将在完成功能更新或错误修复后更新。

## Components

* SSACode - SSA字幕文件的解析核心。
* [TimelineComposite](src\TimelineComposite\TimelineComposite.md) - 时间轴多语言字幕合并工具。
